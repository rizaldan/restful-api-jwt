# RESTful API Backend Stack
Cars list API as Backend Stack
 - [x] Node.js + Express.js
 - [x] JWT (JSON Web Token)
 - [x] MongoDB

JWT User Roles
- [x] admin
- [x] moderator
- [x] user

## Setup and Run
Installing packages

`npm install`

MongoDB Configuration File

`./app/config/db.config.js`

Run

`node server.js`
